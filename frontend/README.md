# CEYLON POS Software Solutions (Pvt) Ltd 🏪

**Advanced Point of Sale & Inventory Management System — v2.0**

React.js + Tailwind CSS | Dark & Light Theme | Mobile Responsive

---

## 🚀 Quick Start

```bash
npm install
npm start
# Opens at http://localhost:3000
```

## 📦 Build for Production

```bash
npm run build
```

---

## 🗂️ All Pages

| Page | Features |
|------|----------|
| **Dashboard** | Revenue charts, stats, low stock alerts, top products |
| **Point of Sale** | Product grid, QR/Barcode scanner, loyalty points, customer select, cart, receipt |
| **Cashier System** | PIN-based login, cashier profiles, session tracking |
| **Inventory** | Product table, add/edit modal, stock status |
| **Customers** | Customer profiles, purchase history, loyalty display |
| **Loyalty Points** | Tier system (Bronze→Platinum), redeem, add bonus points |
| **Suppliers** | Supplier cards, PO creation, payment tracking |
| **Reports** | Charts, transaction table, export |
| **Settings** | Store info, notifications, security, printing, regional |

---

## 🎨 Tech Stack

- **React 18** + **Tailwind CSS**
- **Recharts** — Charts
- **Lucide React** — Icons
- **jsQR** — QR/Barcode scanning (via CDN)

---

## 🌙 Dark / Light Theme

Click the **Sun/Moon button** at the bottom of the sidebar to toggle. Theme preference is saved to localStorage.

---

## 📷 QR / Barcode Scanner

In the POS page, tap **Scan** button → allow camera → point at barcode.  
Works with product barcodes or SKU codes. Manual input fallback also available.
